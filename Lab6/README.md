# Lab 6: Linear Regression

## Objective
To implement and evaluate a **Linear Regression** model for predicting continuous numerical values. [cite_start]The lab demonstrates the workflow of a machine learning project, from initial data exploration to model training and performance evaluation, using housing market data and an ecommerce customer dataset[cite: 1].

## Techniques Applied

* [cite_start]**Exploratory Data Analysis (EDA):** Performed initial data assessments using `info()`, `describe()`, and visual plotting techniques to identify correlations and data distributions[cite: 1].
* [cite_start]**Data Preparation:** Defined feature matrices (X) and target vectors (y) for regression tasks, such as predicting housing prices based on area demographics and ecommerce spending based on customer behavior[cite: 1].
* [cite_start]**Model Training:** Implemented the `LinearRegression` algorithm from the Scikit-Learn library to establish a baseline predictive model[cite: 1].
* **Model Evaluation:** Utilized key regression metrics to quantify model accuracy and error, specifically:
    * [cite_start]**Mean Absolute Error (MAE)** [cite: 1]
    * [cite_start]**Mean Squared Error (MSE)** [cite: 1]
    * [cite_start]**Root Mean Squared Error (RMSE)** [cite: 1]
* [cite_start]**Practical Application:** Applied the regression workflow to the **Ecommerce Customers dataset** to predict the `Yearly Amount Spent` based on factors like membership length and time spent on the app[cite: 1].
